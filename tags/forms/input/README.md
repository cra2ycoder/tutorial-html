## Input

| Property              | Description                             |
| --------------------- | --------------------------------------- |
| Tag name              | input                                   |
| Required attributes   | type, name, value, id                   |
| Optional attributes   | readonly, required, disabled, autofocus |
| Has close tag?        | no                                      |
| Is group tag?         | no                                      |
| Child tags            | no                                      |
| Element type          | inline                                  |
| Has default styles?   | yes                                     |
| Permitted parent tags | can be any                              |

---
